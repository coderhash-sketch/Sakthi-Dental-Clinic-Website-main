# 🦷 Sakthi Dental Clinic – Official Website

A modern, responsive, and user-friendly healthcare website built for **Sakthi Dental Clinic, Hosur**.

This project was developed as part of a client-based web development assignment, focusing on clean UI/UX design, accessibility, responsiveness, and structured healthcare content presentation.

---

## 🌐 Project Overview

Sakthi Dental Clinic is a trusted dental healthcare provider specializing in care for women, children, and families.  

This website reflects:
- Professionalism  
- Warmth and trust  
- Clean and modern healthcare aesthetics  
- Clear and structured navigation  

The website is fully responsive and optimized for mobile, tablet, and desktop devices.
